# Sahadhyayi

Sahadhyayi is a digital library and AI-powered reading companion.

## Quick start
```sh
git clone <REPO_URL>
cd Sahadhyayi
npm install
npm run dev
```

## Environment variables
- VITE_SUPABASE_URL
- VITE_SUPABASE_ANON_KEY
- VITE_SENTRY_DSN
- SUPABASE_URL
- SUPABASE_ANON_KEY
- SUPABASE_SERVICE_ROLE_KEY
- GOODREADS_KEY
- GOODREADS_SECRET
- GOODREADS_CALLBACK_URL
- SUPABASE_EDGE_STT_KEY
- PORT

## How to run
- `npm run test`
- `npm run build`
